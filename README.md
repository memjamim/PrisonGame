# PrisonGame
A wordsearch/maze escape room game

The game is intended to be played between two players: the Inmate and the Guard.

Controls

Enter / LMB / Right Arrow Key to progress dialogue

RMB / Left Arrow Key to go back during dialogue

Keyboard for typing passphrases between stages

Inmate Only:

Wordsearch: Left click and hold to highlight words

Maze: Arrow Keys for movement
